# Conversor 🔢
Calculadora e conversora de bases numéricas desenvolvida em **Java Puro**. O objetivo é realizar operações aritméticas e conversões entre **Binário, Decimal e Hexadecimal** diretamente pelo terminal, sem dependências externas.

##  Funcionalidades

* Conversor decimal -> binário
* Conversor decimal -> hexadecimal
* Conversor binário -> decimal
* Conversor binário -> hexadecimal
* Conversor hexadecimal -> decimal
* Conversor hexadecimal -> binário
